# Virtual Classroom - Promotional Website

This is a modern, comprehensive promotional website for the Virtual Classroom project - a comprehensive learning management system that combines the best features of Google Classroom, WhatsApp, and Zoom.

## Features

- Responsive design that works on all devices
- Modern UI with gradient backgrounds and smooth animations
- Interactive navigation and smooth scrolling
- Comprehensive sections showcasing the platform's capabilities
- Technology stack highlights
- Testimonials from users
- Contact form for inquiries

## Technology Stack

- HTML5
- CSS3 (with Flexbox and Grid)
- JavaScript (ES6+)
- Font Awesome for icons
- Google Fonts (Poppins)

## Sections

1. **Hero Section** - Introduction to the Virtual Classroom platform
2. **Features** - Detailed overview of platform capabilities
3. **About** - Information about the project and its benefits
4. **Technology Stack** - Details of the tech used in development
5. **Demo** - Preview of the platform in action
6. **Testimonials** - Feedback from educators and students
7. **Call to Action** - Encouraging users to try the platform
8. **Contact** - Form and information for reaching out

## How to Run

1. Clone or download this repository
2. Open the `index.html` file in a web browser, or
3. Serve the files using a local server:
   - Using Python: `python3 -m http.server 8000`
   - Using Node.js: `npx serve`
   - Using PHP: `php -S localhost:8000`
